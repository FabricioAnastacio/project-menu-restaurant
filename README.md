## Big Lanches Do Tanjiro 🍔🍟

Este projeto visa ser um cardápio online inovador de fácil usabilidade e intuitive, sua principal característica é a rapidez para fazer o pedido e informações claras, tornando tanto acessível para o cliente quanto para o proprietário do restaurante/delivery.

O nome é de propriedade do delivery o qual fez a solicitação do cardápio, bem como o conteúdo do mesmo que também é de autoria do dono.
